# React-Python-API
React-Python-API


## Brug 
#### Server site:

    mkdir flask-server

Dette vil laver en ny mappe

    cd flask-server
    touch server.py

Installer derefter disse ting på din backend (server site) -> kun på mac:

    python3 -m venv venv 
    source venv/bin/activate
    pip3 install Flask 


#### Client site:

    npx create-react-app client --template typescript
